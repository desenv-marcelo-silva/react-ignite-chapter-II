## Run Locally

Clone the project

```bash
  git clone https://github.com/desenv-marcelo-silva/react-ignite-chapter-II.git
```

Go to the project directory

```bash
  cd react-ignite-chapter-II
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
